# Deep Generative Models Assignment

## Overview
This project implements basic deep generative models for density estimation and data generation. It focuses on autoregressive models and normalizing flows using PyTorch.

---

## Tasks
The assignment includes:

- Training a MADE model on binary MNIST (digits 0 and 1)
- Modelling 2D data using autoregressive Gaussian models
- Extending to mixture of Gaussians
- Training a RealNVP normalizing flow
- Applying reverse KL divergence for training
