## Project README: GloVe Embeddings for Text Classification

This notebook demonstrates the process of training custom GloVe (Global Vectors for Word Representation) embeddings from scratch using a text dataset, visualizing these embeddings, and then utilizing them for a text classification task.

## Project Overview

1.  **Dataset Unzipping**: The `ag_news_csv.zip` dataset, containing news article classifications, is unzipped.
2.  **Library Imports**: Essential Python libraries for data manipulation, deep learning (PyTorch), visualization, and machine learning are imported.
3.  **Dataset Loading**: Training and testing datasets are loaded into Pandas DataFrames.
4.  **Text Normalization**: Custom functions are defined and applied to clean and preprocess the text data, including lowercasing, removing punctuation and numbers, and splitting merged words.
5.  **GloVe Model Training**: A custom GloVe model is built and trained to learn word embeddings. This involves:
    *   Building a vocabulary from the training data.
    *   Constructing a co-occurrence matrix based on word proximity.
    *   Defining and training a PyTorch-based GloVe model with a custom loss function.
6.  **Word Embedding Visualization and Similarity**: The learned GloVe embeddings are visualized in 2D using UMAP (Uniform Manifold Approximation and Projection). Additionally, cosine similarity is used to find words closest to a given sample word, demonstrating the semantic relationships captured by the embeddings.
7.  **Text Classification with GloVe Embeddings**: The learned GloVe embeddings are applied to a text classification task. Sentence embeddings are generated by averaging word embeddings, and a simple neural network classifier is trained and evaluated on the AG News dataset.
